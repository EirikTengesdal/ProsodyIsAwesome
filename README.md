# ProsodyIsAwesome

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains an installation guide for the **Wavelet Prosody Toolkit** for prosodic analysis. The PDF version of the guide is typeset with an adapted version of the **Tufte-Inspired Quarto Extension**.

## Documentation

Please see the [`tufte-inspired/INSTALL_TOOLKIT_README.pdf`](tufte-inspired/INSTALL_TOOLKIT_README.pdf) guide for comprehensive installation instructions for the Wavelet Prosody Toolkit. The guide includes:

- Cross-platform installation instructions (Windows, macOS, Linux)
- Two installation methods:
  - Global user-level installation for general use
  - Per-project installation with isolated dependencies for reproducible research
- Step-by-step setup procedures
- Troubleshooting guidance
